# Student_Marks_Analyzer_With_Python
It is a student marks nalayzer it asks for subjects and there marks and outputs total, percentage, grade ,highest/lowest.

It first ask user how many subject do you have?
then one by one it asks for subject name and marks
Then it displays the total, percemnmatge, grade, highest/lowest .

*Tech Stack:*
  - Python 3
  - CLI 
  - Pure python (No Libraries)
*how to run:*
1. Clone the repository
2. Open it in any code editor and just run it
